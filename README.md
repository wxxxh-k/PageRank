# PageRank
Exploring Non-Uniform Teleportation in Nonlocal PageRank\
PageRank is a widely used algorithm for ranking nodes in networks such as websites, social
networks, and citation networks. Traditional PageRank assumes uniform teleportation,
where a random walker can jump to any node with equal probability. Non-uniform
teleportation, which assigns higher probability to certain nodes, can improve ranking
accuracy and personalization.\
Nonlocal PageRank extends classical PageRank by considering long-range connections, allowing nodes that are not directly connected to influence each other. However,
the combination of non-uniform teleportation with Nonlocal PageRank is largely unexplored.
